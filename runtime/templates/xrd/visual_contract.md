# XRD 视觉合同

## 共同规则

- 页面尺寸、图层留白和标记密度随系列数、标签长度与元素数量自适应，不锁死为单一画幅。
- 字号均按 Origin 的物理 point 合同写入并反读；X/Y 标题、刻度、图例必须使用同一字体族。
- 轴标题、刻度、边框和数据线宽保持论文级层级；预览与 Origin 使用同一冻结 style。
- 图例默认无边框；位置不作为科学成功门禁，允许用户在可编辑 OPJU 中继续微调。
- 不出现源文件没有的峰线、峰标、物相名称、拟合或材料结论。

## 普通扫描

- 单系列为真实强度折线并显示真实 Y 坐标。
- 多系列为清晰堆叠折线；Y 数字在显示变换失去物理意义时隐藏，并在报告中声明归一化与垂直错开。

## Rietveld 精修

- Observed：空心圆点；Calculated：清晰主拟合线。
- Background：弱化辅助线；Difference：按文件中提供的源值直接绘制。
- Phase ticks：使用来源表中的稀疏 X 位置，在图底部按物相分车道显示；仅创建 Y-lane helper，不改 phase X。
- 相位刻线使用 Origin 官方纵向刻线 symbol；多相采用同一审美体系内的可区分颜色。
- 图例无边框；辅助控制列不进入图例。
